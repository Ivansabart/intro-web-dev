1. --- Diferencias entre HTML, CSS y JavaScript ---
HTML: Estructura la información que verá el usuario en el proyecto final.
CSS: Estilos e interacción visual para dar al usuario una buena presentación y experiencia de producto.
Muchos desarrolladores frontend se quejan porque no lo entienden bien.
JavaScript: Lenguaje de programación que dará todo el comportamiento lógico e intercomunica el front con el backend a través de su librería Node.js.
2. --- Tipo de Desarrollador Web ---
Frontend: Se dedica a desarrollar la parte visual que ve el usuario final en el lado del cliente (navegador web).
Usan frameworks CSS, librerías y preprocesadores CSS. Compilador JS. 
Mantenerse actualizado.
Backend: Se dedica a desarrollar el comportamiento de los datos que envía el cliente al lado del servidor.
Utiliza:
  - Lenguajes de programación
  - Frameworks
  - Infraestructura
  - Bases de datos
Fullstack: Maneja tecnologías que usa un front y un backend aunque con una especialización porque es muy extenso conocer todo.
--- Tipos de sitios Web ---
Página estática: Páginas de información, no hay interacción con el cliente, no hay cambios en mucho tiempo. (Landing pages, Blog post)
Página dinámica: Están conectadas a una base de datos, web apps, se puede interactuar, se guardan cambios, cambia en poco tiempo. (Gmail, YouTube)
--- HTML ---
Anatomía de una página web:
HEADER: Puede tener el logo y barra de navegación.
MAIN: Contenedor principal, que contiene la estructura de las secciones.
ASIDE: Contenedor lateral con información.
FOOTER: Se coloca información de contacto y normalmente se coloca abajo. Puede estar en un lateral por si hay un scroll infinito.
